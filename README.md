# EmployeeHub

EmployeeHub is a modern full-stack employee management application.

## Tech Stack

Backend:
- ASP.NET Core 10
- Entity Framework Core
- SQL Server

Frontend:
- Angular
- TypeScript

Security:
- Keycloak
- OpenID Connect
- JWT

Infrastructure:
- Docker
- Docker Compose
- nginx

CI/CD:
- GitHub Actions
